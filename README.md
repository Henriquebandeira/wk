# wk
wk is a simple Wikipedia search, so you can see the wiki basics without wasting time

install
```
pip install wikipedia
```
or
```
pip3 install wikipedia
```

run
```
python3 wk.py [search]
```
help
```
wk [search]

wk -n 5 [search]
                number of lines.

wk -l pt [search]
                language portuguese or out language

wk -n 10 -l jp search
                or
wk -l es -n 1 [search]
```
